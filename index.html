Accello

Accello is Virtusa’s in-house developed Robotic Process Automation (RPA) tool. Accello faciliates users of the both end of the technoogy spectrum to do automation. For business users (operational users), Accello provides extremely user-friendly ‘Record and Replay’ and ‘Drag and Drop’ automation methods. Technical users (power users) can make use of Accello 4GL programming capabilities to create more complex libraries, and advance debugging.

Centroid

Centroid is Virtusa’s in-house developed comprehensive test automation tool that successfully addresses the challenges in conventional test automation. Centroid supports test automation of multiple channels and provides rich features for test execution and management, performance monitoring on mobile devices, and parallel test execution through agents. InsightLive

InsightLive is a code analyser tool developed by virtusa. It enables the developer/ user to fix issues that come up in every module of code that is commited into the repository, by analysing the code commit and isolating the standard code violations. The identified issues and violations will be displayed in a dashboard of several metrics.
